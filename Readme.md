\# P001 - Basic OpenGL C++ : Obstacle Dodger



A modern C++ OpenGL 3.3 Core Profile game demonstrating real-time rendering, shader programming, FreeType text rendering, OpenAL audio integration, collision detection, frame-independent movement, and modular game architecture. This project serves as the first game development milestone in the MSK Real Labs OpenGL learning series.

\# 🎮 Overview



Obstacle Dodger is a modern OpenGL game written entirely in C++. The player controls a circular object that must dodge falling obstacles while the game gradually increases in difficulty.



The project focuses on learning modern graphics programming practices including:



\- Modern OpenGL Rendering

\- Shader Programming

\- VAO/VBO/EBO

\- Frame Independent Movement

\- Collision Detection

\- FreeType Font Rendering

\- OpenAL Audio

\- Modular C++ Design
